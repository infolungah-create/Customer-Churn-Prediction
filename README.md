# Customer Churn Prediction & Revenue Analysis

## Project Overview
This project analyzes customer behavior in an e-commerce dataset to predict churn, forecast revenue, and provide business recommendations.

## Tools Used
- Python (Jupyter Notebook)
- Pandas, NumPy
- Scikit-learn, XGBoost
- SHAP
- Matplotlib, Seaborn

## Key Features
- Customer churn prediction (Logistic Regression, Random Forest, XGBoost)
- Revenue forecasting (Linear Regression, Random Forest Regressor)
- Customer segmentation
- SHAP model explainability

##  Results
- Achieved up to 100% accuracy in churn prediction
- Identified key drivers of churn and revenue
- Developed actionable business strategies

##  Files Included
- Jupyter Notebook (.ipynb)
- Dataset (.csv)
- Report (.docx)

##  How to Run
1. Install required libraries:
   pip install pandas numpy matplotlib seaborn scikit-learn xgboost shap imbalanced-learn

2. Open Jupyter Notebook

3. Run all cells

##  Insights
- High-value customers are less likely to churn
- Discounts and delivery time impact revenue
- Customer engagement drives spending behavior
